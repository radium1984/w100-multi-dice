# W100 Dice

A browser-based dice roller for pen and paper games. Roll 1–6 classic six-sided dice or a single 100-sided die (d100).

## Features

- **d6 mode** — select 1 to 6 dice, see each result with a sum
- **d100 mode** — roll a 100-faced die with a number spinner animation
- Dark mode interface with animated dice
- Sound effect on each roll
- Roll history (last 10 rolls)

## Usage

Download all Files, put them into a Folder and open `index.html` in a browser. Click the dice or press "Roll All".
